Here we have setup the simplest server possible.
  • bin/demo_api_client
  • bin/demo_api_server
  • lib/config.pm
  • lib/DemoApi.pm
By default new servers have two valid api methods, these help with testing and self documentation.
  • methods
  • hello
Test our server: (locally)

./bin/demo_api_client

./bin/demo_api_client hello arg1 val1 arg2 val2

Start the server:

./bin/demo_api_server start
